记录下第一次搭建博客

第一步 环境准备

  1. 访问 Node.js官网 下载并安装 Node.js
  2. 访问 Git官网 下载并安装 Git
  3. 按 win + R 输入 cmd 打开终端验证安装，输入：
  • 看到版本号就安装成功了

第二步 创建GitHub远程仓库

  1. 创建/登录 GitHub 账号（需要魔法）
  2. 点击右上角的 + 号，选择 New repository
  3. Repository name 只能填写 你的用户名.github.io
  4. 选择 Public ，然后点击 Create repository 创建

第三步 创建本地Hexo博客文件夹

  1. 找一个文件夹或者磁盘，右键空白区域选择 Open Git Bash here
  2. 安装 Hexo 脚手架：
