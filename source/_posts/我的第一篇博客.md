## Step 1 创建一个新的博客文件
``` bash
hexo new "博客的名字"
```

## Step 2 本地预览，检查文章排版
``` bash
hexo c && hexo s
```
打开本地网站 (http://localhost:4000) 检查文章排版。

## Step 3 发布文章
``` bash
hexo c && hexo g -d
```
