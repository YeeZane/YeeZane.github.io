## Step 1 写一个新博客
``` bash
hexo new "博客的名字"
```

## Step 2 检查文章排版
``` bash
hexo clean && hexo s
```
本地网站 (http://localhost:4000) 检查文章排版。

## Step 3 发布
``` bash
hexo clean && hexo g && hexo d
```
