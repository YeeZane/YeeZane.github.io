# 自用Markdown写文基础语法表

> 虽然现在各种mk转换工具都很完善齐全，甚至只要给AI一个文档他就能立马给你转换成任何你想要的格式，但我个人认为能够看得懂mk还是很有必要的一件事...

<!-- more -->

| **功能名称** | **语法示例** | **备注** |
| :---: | :---: | :---: |
| **一级标题（H1）** | # 文字 | 一个文章一个H1,注意空格 |
| **二级标题（H2）** | ## 文字 | / |
| **三级标题（H3）** | ### 文字 | / |
| **四级标题（H4）** | #### 文字 | / |
| **五级标题（H5）** | ##### 文字 | / |
| **六级标题（H6）** | ###### 文字 | / |
| **粗体** | \*\*文字** | 或 \_\_文字__ |
| **斜体** | \*文字* | 或 \_文字_ |
| **粗斜体** | \*\*\*文字\*** | 或 \_\_\_文字\___ |
| **删除线** | \~~文字~~ | / |
| **分割线** | --- | 或 *** |
| **无序列表** | * 文字 | 或 - 文字 / + 文字，注意空格 |
| **有序列表** | 1. 文字 | 注意空格 |
| **引用** | > 文字 | 注意空格 |
| **代码块** | \~~~python<br>print("Hello")<br>~~~ | 或 \```python<br>print("Hello")<br>``` |
| **行内代码** | \`code` | / |
| **行内显示单反引号** | \`\` \`main()` `` | \``单反引号标注的函数名`` |
| **链接** | \[链接名称](url) | \[YeeZane's blog](https://yeezane.github.io/) |
| **图片** | !\[图片描述](url/image.jpg) | 链接的语法前加个! |
| **强制换行** | 行尾加两个空格 | / |
